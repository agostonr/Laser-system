◆ FindLaserRoot()

Transform Laser.Logic.Query.IQueryableLaserActor.FindLaserRoot ( )

Finds the compound Laser Actor's root. Put this on e.g. a laser emitter cube so that when moving the cube the emitter doesn't fall apart.

Relatively expensive to execute so make sure to cache the result at startup.

Returns
the root of the laser actor as marked by the developer or null if no such script is added to any of the parents.

Implemented in Laser.Logic.BlockingLaserReceiver, Laser.Logic.LaserEmitter, and Laser.Logic.LaserRelay.
